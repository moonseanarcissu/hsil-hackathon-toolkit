# HSIL Hackathon 2026 — Taiwan Hub

這是 HSIL Hackathon 2026 台灣站的 Claude Code 開源工具包。
由聯合通科技 (Cloud Communication Technology Ltd.) 簡承哲專案經理製作，供所有參賽者使用。

## 比賽資訊

- **日期**: 2026/4/10-11（週五、六）
- **地點**: 台大公衛學院
- **主題**: 運用 AI 打造高價值醫療體系 (Building High-Value Health Systems: Leveraging AI)
- **簡報**: 3 分鐘 Pitch + 1.5 分鐘 Q&A
- **評分**: 10 項標準，每項 0/1/2 分，共 20 分

## 可用 Skills（輸入 `/` 即可看到全部指令）

### 研究階段
| 指令 | 用途 |
|------|------|
| `/brainstorm` | 腦力激盪 — 探索 11 個參考方向，HMW 問題發想，概念評估 |
| `/deep-research` | 深度研究 — 5 階段結構化健康議題研究 |
| `/user-persona` | 使用者人物誌 — 建立 Persona、旅程地圖、同理心地圖 |

### 分析階段
| 指令 | 用途 |
|------|------|
| `/market-analysis` | 市場分析 — TAM/SAM/SOM + 競品分析 + Porter's 5 Forces |
| `/swot-canvas` | SWOT 與商業畫布 — 策略分析 + 商業模式設計 + 單位經濟 |
| `/regulatory-check` | 法規檢查 — 台灣健康科技法規合規評估 |
| `/tech-arch` | 技術架構 — AI 技術選型 + 系統架構圖 + 台灣資料源盤點 |

### 簡報階段
| 指令 | 用途 |
|------|------|
| `/pitch-deck-guide` | 投影片引導 — 8 頁投影片逐頁製作教練 |
| `/data-story` | 數據故事力 — 將數據轉化為有說服力的視覺故事 |
| `/pitch-timer` | 簡報計時 — 字數分析、時間控制、講稿優化 |

### 演練階段
| 指令 | 用途 |
|------|------|
| `/pitch-simulation` | 模擬評審 — 10 項標準打分 + 三種評審視角 + 模擬 Q&A |
| `/qa-prep` | 問答準備 — 預測評審問題 + ARC 回答框架 + 團隊分工 |

## 建議工作流程

```
Phase 1: 選題與研究
  /brainstorm       → 探索方向、發想概念
  /deep-research    → 深入研究選定議題
  /user-persona     → 建立使用者人物誌

Phase 2: 分析與設計
  /market-analysis  → 市場數據與競品分析
  /swot-canvas      → SWOT + 商業模式
  /tech-arch        → 技術架構設計
  /regulatory-check → 法規合規檢查

Phase 3: 簡報製作
  /pitch-deck-guide → 建立 8 頁投影片
  /data-story       → 數據視覺化與故事力

Phase 4: 演練
  /pitch-timer      → 計時練習
  /pitch-simulation → 模擬評審打分
  /qa-prep          → Q&A 準備
```
